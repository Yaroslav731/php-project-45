# Brain games (Игры разума)

[![Actions Status](https://github.com/Yaroslav731/php-project-45/actions/workflows/php-linter.yml/badge.svg)](https://github.com/Yaroslav731/php-project-45/actions)

### Описание
«Игры разума» — это набор из пяти консольных игр, построенных по принципу популярных математических опросников. Каждая игра задает пользователю три вопроса. Если ответы на все вопросы верны, игра считается пройденной.

### Список игр:
1. Калькулятор: Решите случайное математическое выражение.
2. Проверка на четность: Определите, является ли число четным.
3. НОД: Найдите наибольший общий делитель двух чисел.
4. Прогрессия: Найдите пропущенное число в арифметической прогрессии.
5. Простое число: Определите, является ли число простым.
